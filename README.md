# node-crud-assignment
A comprehensive backend demonstration assignment built using **Node.js** and **Express.js**. This project features fully functional CRUD (Create, Read, Update, Delete) operations mapped across multiple database engines independently, highlighting flexible architectural connectivity layer integrations.

---

## 🛠️ Core Databases Integrated
* **MongoDB Backend Configuration:** Handles object-data document management streams using standard ODM frameworks.
* **MySQL Relational Configuration:** Executes relational structured query tables and dataset manipulations.
* **SQLite Lightweight Instance:** Manages standalone embedded localized relational file storage environments using `students.db`.

💻 Local Execution Guide
Open your workspace terminal and install the project module packages:

Bash
npm install
Run any database CRUD operation script independently using node runtime environment hooks:

Bash

# OR
node mysql.js
